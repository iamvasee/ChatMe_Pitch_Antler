1. **Clone Repository**:
   ```bash
   git clone https://github.com/iamvasee/ChatMe_Pitch_Antler.git
   cd ChatMe_Pitch_Antler
   ```

2. **Serve Locally**:
   ```bash
   # Using Python
   cd public

   # Using Node.js
   npx serve public

   php -S localhost:8000 -t public
   ```

3. **Open Browser**: Visit `http://localhost:8000`
